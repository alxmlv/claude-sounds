# Claude Sounds - Warcraft Peon Edition

Your Claude Code assistant now responds with classic Warcraft peon voice lines.

## Install

```bash
git clone <repo-url> && cd claude-sounds && ./install.sh
```

## Sound Mapping

| Hook | Sound | File |
|---|---|---|
| SessionStart | "Ready to work" | PeonReadyToWork.wav |
| UserPromptSubmit | Random: "Okeydokey!" / "Work, work" | PeonYes4.wav, PeonWorkWork.wav |
| Notification | "Something need doing?" | PeonSomethingNeedDoing.wav |
| Stop | "Job's done!" | PeonJobsDone.wav |

## Uninstall

```bash
./uninstall.sh
```
